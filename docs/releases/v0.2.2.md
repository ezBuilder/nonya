# nonya v0.2.2

오탐 알림을 줄이는 품질 패치입니다.

## Fixes

- `--all --mode auto`에서 `<<DONE>>` 없이 끝난 세션을 더 진행시킬 수 있는지 확인할 때, 대상 창이나 pane을 안전하게 특정하지 못하면 "자동으로 깨우지 못했다" 알림을 띄우지 않습니다.
- 실제 주입 가능한 tmux/GUI 대상은 기존처럼 깨우되, 불확실한 대상은 조용히 재시도합니다.
- unreachable keep-going 추정 상태를 `alert-only` ledger 항목으로 쌓지 않아 오래된 세션이 품질 알림처럼 보이는 문제를 막았습니다.
- macOS 메뉴는 유휴 세션을 활성/확인 필요 목록에 표시하지 않습니다.
