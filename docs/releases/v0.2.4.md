# nonya v0.2.4

오탐 방지 릴리즈입니다. 자율 모드가 이제 정상 완료/유휴 세션을 `<<DONE>>` 미출력만으로 깨우지 않습니다.

## Fixes

- `done` 상태가 됐지만 `<<DONE>>`을 출력하지 않은 모든 세션을 무조건 keep-going으로 보던 동작을 제거했습니다.
- 최근 사용자 지시에 `<<DONE>>` 완료 계약이 있을 때만 미완료 작업으로 보고 계속 진행합니다.
- Claude/Codex 양쪽에서 사용자 메시지만 계약 신호로 읽고, 도구 출력이나 문서 안의 `<<DONE>>` 문자열은 무시합니다.
- 단일 세션 루프와 Watch all 스캐너 모두 같은 기준을 사용합니다.
- macOS 메뉴 문구를 "요청된 `<<DONE>>` 작업만 계속"으로 바꿔 실제 동작과 맞췄습니다.
