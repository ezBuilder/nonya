# nonya v0.2.3

GUI 앱 복구가 composer에 문장만 남기고 전송하지 못하는 문제를 고친 품질 패치입니다.

## Fixes

- macOS GUI 앱 주입에서 기본 제출 동작을 `Return` 후 `Command+Return` fallback으로 보강했습니다.
- Claude/Codex 계열 Electron/WebKit composer가 Return을 줄바꿈 또는 편집 동작으로 처리하는 경우에도 메시지가 실제 제출되도록 했습니다.
- disposable `NonyaProbe` 라이브 테스트가 이제 "문자가 입력됐는가"가 아니라 "제출됐는가"를 검증합니다.
- 다중창 GUI 게이트는 계속 키 0개를 보장합니다.
